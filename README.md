### 🚀 Getting Started

#### **Development Environment Setup**
```bash
# Prerequisites
node >= 18.0.0
react-native-cli
xcode >= 14.0 (iOS)
android-studio >= 2022.3 (Android)
docker >= 20.0

# Quick start
git clone https://github.com/edwson/pawsroam.git
cd pawsroam

# Install all dependencies
npm run bootstrap

# Environment configuration
cp .env.example .env
# Configure API keys for Google Maps, OpenAI, Stripe, etc.

# Start development servers
npm run dev:all
```

#### **Platform-Specific Development**
```bash
# iOS development
npm run ios:dev
npm run ios:build
npm run ios:test

# Android development  
npm run android:dev
npm run android:build
npm run android:test

# Web development
npm run web:dev
npm run web:build
```
